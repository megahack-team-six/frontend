# JusBras

## Instalação de dependências
```
yarn install
```

### Compila e roda o servidor de desenvolvimento com hot-reload
```
yarn serve
```

### Compila e minifica para produção
```
yarn build
```

### Roda o lint e corrige erros
```
yarn lint
```
